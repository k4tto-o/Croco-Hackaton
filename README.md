# Hackaton
Python hackaton
> Idea:
   App o Pagina WEB con retos diarios y semanales ecologicos.
```
   Sistema de recompensas:
     10 puntos por cada logro, los puntos se acumulan y cada semana se canjean por (...) recompensa.
   Seguimiento del progreso:
     Poco a poco (segun el avance del usuario) se formaria un mapa donde se presentan las acciones ecologicas del usuario, se actualizaria mensual o semanalmente.
   Adicional:
     - Sistema de consejos o recomendaciones cada que se pierda un dia de la racha.
     - Sistema comunitario para compartir los logros y el mapa del progreso.
     - Bot de discord que facilite el acceso a los retos y al mapeo del progreso.
